Embedding 3js-robot

Each widget below is a self-contained <robo-sim> element. They run independently in the same page.

1. Click-to-reach arm (full controls)

Turn on Reach mode and click a point in the arm's plane.

2. A minimal embed (no panel, fixed scene)

Just the view - good for an inline illustration in an article.

How to embed elsewhere

<script type="module" src="robo-sim.js"></script>
<robo-sim scene="reacher" base="/path-to/" style="height:440px"></robo-sim>

Build the standalone file with npm run build:embed (outputs dist-embed/robo-sim.js), and host the scenes/ folder at the base URL you pass.
